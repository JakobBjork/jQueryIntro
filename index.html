Om jQuery

Vad är jQuery

jQuery är ett litet och snabbt JavaScript bibliotek med många funktioner. Med en väl utvecklad API mycket möjligheter och bra stöd för ytterligare tillägg har jQuery ändrat hur miljoner skriver JavaScript.

Historia

jQuery skapades i Januari 2006 av John Resig. Det underhålls I nuläget av team med utvecklare under Timmy Willison. Från slutet av 2006 och fram till 2012 fanns det två olika licenser för det. Något som orsakat viss förvirring. jQuery växer stadigt. 2015 användes det av 63% av dom 1 miljon mest besökta sidorna. 2018 var den siffran uppe I 73% och 22,4% av alla webbsidor.

Varför jQuery

Syftet med jQuery är att underlätta användandet av JavaScript. Många vanliga JavaScript uppgifter kan dras ner från flera rader till en ensam rad. Även mer avancerade JavaScript saker som AJAX och DOM förenklas. Förutom att ha bra färdigt stöd för följande så finns det även många färdiga tillägg man kan installera.
jQuery är trots att det finns många konkurrenter störst och används av stora företag som Google, Microsoft och Netflix.

jQuery kan användas till

  • HTML och DOM manipulering
  • CSS manipulering
  • Manipulering av händelser i HTML
  • Grafiska effekter och animeringar
  • AJAX
  • Verktyg
